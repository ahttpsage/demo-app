## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm, yarn, pnpm, or bun

### Installation

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Building for Production

```bash
npm run build
npm run start
```

## Deploy on Vercel

The easiest way to deploy this app is using [Vercel](https://vercel.com):

1. Push your code to a Git repository (GitHub, GitLab, or Bitbucket)
2. Import your repository on [Vercel](https://vercel.com/new)
3. Vercel will automatically detect Next.js and configure the build settings
4. Click "Deploy" and your site will be live!

Alternatively, install the Vercel CLI:

```bash
npm i -g vercel
vercel
```

## Customization

### Update Personal Information

Edit `app/page.tsx` to customize:
- Name and title
- About section content
- Services offered
- Technical skills
- Contact links

### Styling

Modify colors and themes in:
- `app/globals.css` - Global styles and animations
- `app/page.tsx` - Component-specific Tailwind classes

### Add New Sections

Simply add new section components in `app/page.tsx` following the existing pattern.
